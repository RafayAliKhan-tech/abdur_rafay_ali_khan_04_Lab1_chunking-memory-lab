# Advanced Memory Chunking Lab (HCI Project)

## 🧠 Overview
This is an interactive web-based experiment demonstrating **Miller's Law** and **chunking in short-term memory**.  
Users can test their memory using different real-world data types, such as **Phone Numbers, ATM PINs, CNIC numbers, and Website UI elements**.

The project highlights how **chunking improves memory recall** and reduces cognitive load, which is a key concept in **Human-Computer Interaction (HCI)**.

---

## 🔥 Features
- Select **data type**: Phone, ATM PIN, CNIC, Website UI  
- Choose **Ungrouped vs Chunked** test  
- **Dynamic random data** generation for realistic difficulty  
- **5-second countdown timer**  
- **Accuracy score** calculation (%)  
- **Responsive and modern UI** (glassmorphism + animations)  

---

## 🎯 How It Works
1. User selects the type of data from the dropdown.  
2. Click **Ungrouped** or **Chunked** test.  
3. Data is displayed for **5 seconds** with a countdown timer.  
4. After the timer ends, user types what they remember.  
5. System calculates **correct digits and accuracy**, showing the result.
